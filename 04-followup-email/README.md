# Follow-Up Email from Call Notes

**30 seconds, not 15 minutes.**

## What This Does

Takes your messy call notes and turns them into a specific follow-up email that references actual discussion points, includes agreed next steps with owners and dates.

## What You Need

- **Claude Code** installed ([guide](https://docs.anthropic.com/en/docs/claude-code))
- Claude Code installed, your call notes (bullet points, messy text, or a transcript)

## How to Get the Input File

Just paste your notes or point Claude Code to a notes file. No export needed.

## The Prompt

Copy this into Claude Code:

```
Turn these call notes into a follow-up email.

Reference specific things we discussed.
Include the next steps we agreed on with owners and dates.
Keep it under 150 words.
Tone: professional but warm.

[paste your notes here or point to notes.txt]
```

See [prompt.md](./prompt.md) for the copy-pasteable version.

## What You'll Get Back

An email draft with a subject line and body that references their actual pain points, mentions specific features discussed, and lists next steps with owners and dates.

## How to Verify

Read the draft. Does it accurately reflect what was discussed? Fix the tone if needed. Edit any details that are off. Then send.

## Time Saved

| Manual | Claude Code |
|--------|-------------|
| 15 min of drafting | 30 seconds + quick edit |
