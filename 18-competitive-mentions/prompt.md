# Competitive Mention Tracker — Prompt

Copy-paste this into Claude Code.

```
Read all transcripts in /calls/.

Scan for competitor mentions. For each mention extract:
- Which competitor
- What was said
- Who said it (prospect or rep)
- Context of the conversation
- How the rep handled it
- Outcome of that segment

Group by competitor.
Show frequency and common objections per competitor.
Output as competitive_intel.md.
```

## Customization

- Adjust criteria to match your sales process
- Change output format if needed (CSV, JSON, markdown)
- Add or remove fields based on what is useful for your workflow
